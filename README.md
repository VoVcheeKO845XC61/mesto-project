Место — Социальная сеть для фотографий

Проектная работа для курса «Фронтенд-разработчик» платформы Яндекс Практикум.

Описание

Место — это одностраничное веб-приложение, которое позволяет пользователям делиться фотографиями, редактировать свой профиль и взаимодействовать с контентом других пользователей. Приложение демонстрирует современные подходы к фронтенд-разработке, включая работу с REST API, управление состоянием, валидацию форм и адаптивный дизайн.

Основные возможности

- **Управление профилем:** Редактирование имени, описания и аватара пользователя
- **Галерея карточек:** Просмотр фотографий, загруженных пользователем и другими пользователями
- **Система лайков:** Возможность ставить и убирать лайки на карточки с отображением количества лайков
- **Добавление новых карточек:** Загрузка собственных фотографий с названием и ссылкой
- **Удаление карточек:** Удаление собственных карточек (доступно только для автора)
- **Просмотр информации о карточке:** Детальная информация о карточке, включая дату создания и список пользователей, поставивших лайк
- **Валидация форм:** Проверка корректности введённых данных в реальном времени
- **Адаптивный дизайн:** Оптимизация интерфейса для различных размеров экранов


API

Приложение использует REST API сервера Яндекс Практикума для работы с данными:

- `GET /users/me` — получение информации о текущем пользователе
- `GET /cards` — получение списка всех карточек
- `PATCH /users/me` — обновление информации профиля
- `PATCH /users/me/avatar` — обновление аватара
- `POST /cards` — создание новой карточки
- `DELETE /cards/{cardId}` — удаление карточки
- `PUT /cards/likes/{cardId}` — добавление лайка
- `DELETE /cards/likes/{cardId}` — удаление лайка
